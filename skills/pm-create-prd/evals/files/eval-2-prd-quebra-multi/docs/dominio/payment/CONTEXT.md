# Payment

Contexto responsável por processar pagamentos, créditos e estornos.

## Linguagem

**Estorno**:
Devolução do valor pago ao cliente após o cancelamento de um pedido.
_Evitar_: reembolso, refund

**Crédito**:
Valor pré-pago ou promocional que o cliente pode abater em um pagamento futuro.
_Evitar_: saldo, cashback
